# Gestor-Videoclub

Proyecto de ejemplo "VideoClub" — gestor para un videoclub. Este repositorio fue importado desde el entorno local.

Repositorio remoto: https://github.com/MugenSushi/Gestor-Videoclub

## Cómo contribuir

- Clona el repositorio.
- Revisa las ramas y crea una rama de trabajo nueva para tus cambios.

Ejemplo rápido:

```powershell
git clone https://github.com/MugenSushi/Gestor-Videoclub.git
cd Gestor-Videoclub
git checkout -b feature/mi-cambio
```
